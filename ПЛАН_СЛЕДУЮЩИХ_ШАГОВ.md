# План следующих шагов

## Сделать в первую очередь

- [ ] Загрузить реальный архив Telegram-канала в бота
  - Открыть бота @Alexey_Li_copy_bot
  - Нажать скрепку 📎
  - Выбрать файл архива (result.json или .md) — переносить в папку не нужно
  - Бот проанализирует посты и запомнит твой стиль
  - После этого все генерируемые посты будут в твоём настоящем стиле

## Как выгрузить архив из Telegram (если ещё не сделал)

1. Открой Telegram Desktop (на компьютере, не на телефоне)
2. Перейди в свой канал
3. Три точки (···) → Экспорт истории чата
4. Выбери: только текст, формат JSON
5. Нажми Экспорт — появится файл result.json

## После загрузки архива

- [ ] Проверить кнопку «✍️ Написать пост» — написать пост в своём реальном стиле
- [ ] Проверить кнопку «📋 Контент-план» — создать план на месяц
- [ ] Пополнить баланс на OpenRouter для полноценной работы
  - Зайти на openrouter.ai/settings/keys
  - Добавить $5 — хватит на ~500-1000 постов
